Выпуск пользовательской оболочки Unity 7.6.0

Разработчики проекта Ubuntu Unity, развивающего неофициальную редакцию Ubuntu Linux с рабочим столом Unity, опубликовали выпуск Unity 7.6.0 , который отмечен как первый значительный выпуск за 6 лет после прекращения развития оболочки компанией Canonical. Оболочка Unity 7 основана на библиотеке GTK и оптимизирована для эффективного используемого вертикального пространства на ноутбуках с широкоформатными экранами. Код распространяется под лицензией GPLv3. Готовые пакеты сформированы для Ubuntu 22.04. https://www.opennet.ru/opennews/art.shtml?num=57114
