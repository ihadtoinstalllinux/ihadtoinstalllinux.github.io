Новый выпуск дистрибутива Radix cross Linux

Доступны новые сборки дистрибутива Radix cross Linux, построенного с использованием собственной системы сборки Radix.pro, упрощающей формирование дистрибутивов для встраиваемых систем. Система сборки включает набор Make-файлов и скриптов, написанных на языках Bash и Perl, которые предоставляют средства для работы с архивами исходных текстов входящих в состав дистрибутива компонентов, сборки дистрибутива, управления пакетами и установки.
